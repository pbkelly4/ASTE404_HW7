Detailed Description

template<typename T>
class _vec3< T >

A template class for 3D vector operations.

This class represents a 3D vector with components of type T and supports basic vector operations such as addition, subtraction, dot product, and magnitude.

Template Parameters
TThe type of the vector components (e.g., double).
